替换软件源方法：首先备份原来的软件源配置：

sudo mv /etc/apt/sources.list /etc/apt/sources.list.old

创建新的配置文件，内容为下文所述的源列表：

sudo vim /etc/apt/sources.list
# 添加下文所述软件源列表

然后依次：

sudo apt-get update
sudo apt-get upgrade
