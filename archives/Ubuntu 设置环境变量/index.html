Ubuntu 设置环境变量

熊猫小A 2018-04-28 in 日常技巧

当前终端

export VAR_NAME=value

当前用户

sudo gedit ~/.bashrc

添加对应变量后，source ~/.bashrc

所有用户

sudo gedit /etc/profile

添加变量后，source /etc/profile

PATH 变量

# 加到PATH末尾
export PATH=$PATH:/path/to/your/dir

# 加到PATH开头
export PATH=/path/to/your/dir:$PATH
